CAD File naming convention:

Parts: 	3 Letter code for assembly or subassembly that it belongs to. ex: DIF = Differential
	- Dash
	P for part
	3 Numbers representing part number. ex: 001 is the part #1

	Ex: DIF-P001

Assemblies:	3 Letter code for assembly or subassembly that it represents. ex: DIF = Differential
		- Dash
		A for assembly
		3 Numbers representing assembly number. The  main assembly is 000 and all subsequent subassemblies start at 001

		EX: DIF-A001 is a subassembly of DIF-A000

Main Assembly:	If project is for ex. the Drivetrain of a car, the master assembly will be named DVT-A000
